# Contato Email

Formulário de contato simples com campos de nome, e-mail e mensagem, com envio direto para e-mail via FormSubmit, sem necessidade de backend ou banco de dados.

obs: precisa adicionar um email da sua preferencia no código para enviar o email

## Tecnologias

- [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML) — estruturação do formulário
- [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) — estilização da página
- [FormSubmit](https://formsubmit.co/) — processamento e envio dos dados por e-mail
